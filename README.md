# Scalable Recommendation Engine

## Overview
A hybrid movie recommendation system using collaborative filtering, content-based filtering, and cold start handling.

## Features
- Collaborative filtering (user similarity)
- Content-based filtering (TF-IDF on genres)
- Hybrid recommendation system
- Cold start handling using weighted popularity
- Evaluation using RMSE

## Dataset
MovieLens Latest Small Dataset (100K+ ratings)

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn

## How to Run
cd src  
python hybrid.py
